# Credit Card Fraud Detection

Machine Learning project to detect fraudulent credit card transactions using classification algorithms and data analysis.

## Tech Stack

- Python
- Pandas
- Scikit-learn
- Flask
- HTML/CSS
- Render

## Features

- Fraud transaction prediction
- Data preprocessing & scaling
- Imbalanced dataset handling
- Real-time prediction system
- Flask integration
- Deployed on Render

## Dataset

Dataset used:
https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud

Place the dataset inside:

data/creditcard.csv
